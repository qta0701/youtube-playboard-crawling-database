# YouTube Shorts 대본 추출기

구글 시트에서 YouTube Shorts 영상 ID를 읽어와서 자동으로 대본을 추출하는 GUI 프로그램입니다.

## 주요 기능

- 🎯 **시트 선택**: 5개 시트 중 원하는 시트 선택
- 📊 **실행 모드**: 전체 추출 또는 마지막 대본 이후 추출
- 🔄 **50개 단위 배치 처리**: 안정적인 대량 처리
- 📈 **실시간 진행상황**: 프로그레스바와 로그로 진행상황 확인
- 📋 **결과 집계**: 성공/실패 통계와 소요시간 표시
- ❌ **오류 처리**: 실패한 영상 자동 건너뛰기

## 파일 구성

- `GUI_Extract.py`: GUI 버전 메인 프로그램
- `Main_Extract.py`: 콘솔 버전 프로그램
- `build_exe.py`: 실행파일 생성 스크립트
- `build_exe.bat`: 실행파일 생성 배치파일
- `requirements.txt`: 필수 패키지 목록

## 설치 및 실행

### 방법 1: Python으로 직접 실행

1. **필수 패키지 설치**:
   ```bash
   pip install -r requirements.txt
   ```

2. **GUI 버전 실행**:
   ```bash
   python GUI_Extract.py
   ```

3. **콘솔 버전 실행**:
   ```bash
   python Main_Extract.py
   ```

### 방법 2: 실행파일 생성

1. **`build_exe.bat` 더블클릭** 또는 명령창에서:
   ```bash
   python build_exe.py
   ```

2. **생성된 실행파일 사용**:
   - `dist/YouTube_Shorts_대본추출기.exe` 실행
   - '구글 서비스 계정 JSON키' 폴더 함께 복사 필요

## 사용 방법

### GUI 버전
1. 프로그램 실행
2. **시트 선택**: 드롭다운에서 처리할 시트 선택
3. **실행 모드 선택**:
   - **전체 추출**: 10행부터 모든 영상 처리
   - **마지막 대본 이후**: 기존 대본 데이터 이후부터 처리
4. **"대본 추출 시작"** 버튼 클릭
5. 진행상황 확인 및 결과 집계 창에서 결과 확인

### 콘솔 버전
1. 프로그램 실행
2. 시트 번호 선택 (1-5)
3. 실행 모드 선택 (A/B)
4. 자동 처리 및 결과 확인

## 시트 구조 요구사항

- **'영상 ID'** 열: YouTube 영상 ID 또는 URL
- **'대본내용'** 열: 추출된 대본이 저장될 열
- **10행부터**: 실제 데이터 시작 행

## 기술 사양

- **언어**: Python 3.7+
- **GUI**: tkinter (내장)
- **비동기 처리**: asyncio, aiohttp
- **Google Sheets**: gspread, google-auth
- **50개 단위**: 배치 처리로 안정성 확보
- **재시도 로직**: 실패 시 최대 3회 재시도

## 설정 파일

`GUI_Extract.py` 내부의 경로 설정:
```python
self.CREDENTIALS_PATH = r"구글 서비스 계정 JSON키 경로"
self.SHEET_URL = "구글 시트 URL"
```

## 문제 해결

### 자주 발생하는 오류

1. **"구글 시트 인증 실패"**
   - 서비스 계정 JSON 키 파일 경로 확인
   - 구글 시트 공유 설정 확인

2. **"영상 ID 열을 찾을 수 없음"**
   - 시트에 '영상 ID' 텍스트가 포함된 헤더 확인

3. **"대본내용 열을 찾을 수 없음"**
   - 시트에 '대본내용' 텍스트가 포함된 헤더 확인

4. **"자막을 사용할 수 없음"**
   - 해당 영상에 자막이 없거나 비공개 영상
   - 자동으로 건너뛰고 다음 영상 처리
