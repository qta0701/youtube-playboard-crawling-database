# Claude Code 가이드라인

## 권한 설정

이 프로젝트는 자동 실행 모드로 설정되어 있습니다:

- **파일 편집 (Edit)**: 자동 승인
- **파일 작성 (Write)**: 자동 승인
- **파일 읽기 (Read)**: 자동 승인
- **명령 실행 (Bash)**: 자동 승인

사용자의 승인 없이 모든 파일 작업과 명령 실행이 자동으로 진행됩니다.

## 설정 위치

권한 설정은 `.claude/settings.json` 파일에서 관리됩니다:

```json
{
  "permissions": {
    "defaultMode": "acceptEdits",
    "allow": [
      "Edit:*",
      "Write:*",
      "Read:*",
      "Bash:*"
    ]
  }
}
```

## 작업 방식

Claude Code는 다음과 같이 동작합니다:

1. 파일을 읽거나 편집할 때 별도의 확인 없이 즉시 실행
2. 새 파일을 생성할 때 자동으로 진행
3. 터미널 명령을 실행할 때 자동으로 진행
4. 작업 완료 시 알림 표시 (notify.ps1 훅 사용)
