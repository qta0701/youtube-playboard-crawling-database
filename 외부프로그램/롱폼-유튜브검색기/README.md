# YouTube 검색 및 구글 시트 연동 프로그램

YouTube Data API를 사용하여 키워드로 영상을 검색하고, 결과를 구글 시트에 자동으로 추가하는 프로그램입니다.

## 주요 기능

- YouTube Data API를 통한 영상 검색
- GUI를 통한 직관적인 검색 설정
- 구글 시트에 검색 결과 자동 추가 (벌크 업데이트)
- 영상 메타데이터 자동 추출 (조회수, 좋아요, 댓글, 채널 정보 등)
- 헤더 자동 매칭 (넘버링 및 공백 무시)

## 파일 구조

```
유튜브 검색기/
├── Main_Search.py          # YouTube API 검색 및 데이터 추출
├── GUI_Interface.py        # GUI 및 구글 시트 관리
├── main.py                 # 메인 실행 파일
├── requirements.txt        # 필요한 패키지 목록
├── google_service_key/     # 구글 인증 파일 디렉토리

│   └── client_secret_*.json
└── README.md              # 사용 설명서
```

## 설치 방법

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 구글 클라우드 플랫폼 설정

#### A. 서비스 계정 생성 (구글 시트 접근용)

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속
2. 프로젝트 생성 또는 선택
3. "IAM 및 관리자" → "서비스 계정" 메뉴로 이동
4. "서비스 계정 만들기" 클릭
5. 서비스 계정 생성 후 JSON 키 다운로드
6. 다운로드한 파일을 `google_service_key/service-account-key.json`으로 저장

#### B. OAuth 2.0 클라이언트 ID 생성 (YouTube API 접근용)

1. Google Cloud Console의 "API 및 서비스" → "사용자 인증 정보" 메뉴로 이동
2. "사용자 인증 정보 만들기" → "OAuth 클라이언트 ID" 선택
3. 애플리케이션 유형: "데스크톱 앱" 선택
4. JSON 파일 다운로드
5. 다운로드한 파일을 `google_service_key/` 디렉토리에 저장

#### C. YouTube Data API v3 활성화

1. Google Cloud Console의 "API 및 서비스" → "라이브러리" 메뉴로 이동
2. "YouTube Data API v3" 검색 및 활성화

### 3. 구글 시트 권한 설정

1. 서비스 계정 이메일 주소 확인 (예: `youtube-transcript-extractor@shorts-transcripts-python.iam.gserviceaccount.com`)
2. 구글 시트를 열고 서비스 계정 이메일에 "편집자" 권한 부여

## 사용 방법

### 1. 프로그램 실행

```bash
python main.py
```

### 2. GUI에서 검색 설정

- **검색 키워드**: 검색할 YouTube 키워드 입력
- **최대 결과 수**: 가져올 영상 개수 (1~50)
- **정렬 순서**:
  - `relevance`: 관련성 순
  - `date`: 최신순
  - `viewCount`: 조회수 순
  - `rating`: 평점 순
  - `title`: 제목 순
- **영상 길이**:
  - `any`: 모든 길이
  - `short`: 4분 이하
  - `medium`: 4~20분
  - `long`: 20분 이상
- **시트 이름**: 데이터를 추가할 시트 탭 이름 (기본: "키워드 검색결과")

### 3. 검색 및 시트에 추가

"검색 및 시트에 추가" 버튼 클릭

## 구글 시트 구조

### 헤더 (1행)

시트의 1행은 헤더로, 다음과 같은 컬럼들이 있습니다:

1. 영상 ID
2. 영상 업로드날짜
3. 업데이트 날짜
4. 검색 키워드
5. 영상 링크
6. 제목
7. 채널명
8. 조회수
9. 숏폼여부
10. 영상길이
... (총 61개 컬럼)

### 데이터 (10행부터)

- 10행부터 실제 데이터가 추가됩니다
- A열(영상 ID)을 기준으로 마지막 행을 찾아 새로운 데이터를 추가합니다
- 헤더 넘버링(예: "1. 영상 ID")은 자동으로 무시되고 텍스트만 매칭됩니다

## YouTube API에서 제공하는 데이터

프로그램은 다음 데이터를 자동으로 추출합니다:

### 영상 정보
- 영상 ID, 제목, 링크
- 업로드 날짜
- 조회수, 좋아요 수, 댓글수
- 영상 길이
- 숏폼 여부 (60초 이하)
- 카테고리 ID
- 디스크립션
- 해시태그
- 썸네일 링크

### 채널 정보
- 채널명, 채널 ID
- 채널 링크
- 구독자수
- 채널 국가
- 채널 개설일
- 채널 전체 조회수
- 영상 개수

### 계산 지표
- 구독자 대비 조회수 배율
- 조회수 대비 좋아요/댓글 비율
- 일평균 조회수
- 영상당 평균 조회수
- 조회수 기준 필터 (100만, 500만, 1000만 이상)
- 좋아요 3% 이상 여부

## 주의사항

1. **YouTube API 할당량**: YouTube Data API는 일일 할당량이 있습니다 (기본 10,000 단위). 검색 1회당 약 100 단위를 소비합니다.

2. **OAuth 인증**: 처음 실행 시 브라우저가 열리며 Google 계정 로그인이 필요합니다.

3. **시트 권한**: 서비스 계정에 구글 시트 편집 권한이 부여되어야 합니다.

4. **최대 결과 수**: YouTube API는 한 번에 최대 50개의 결과만 반환할 수 있습니다.

## 문제 해결

### "YouTube API가 초기화되지 않았습니다" 오류

- OAuth 클라이언트 파일 경로를 확인하세요
- 브라우저에서 Google 계정 인증을 완료하세요

### "구글 시트 연결 실패" 오류

- 서비스 계정 JSON 파일 경로를 확인하세요
- 구글 시트 URL이 정확한지 확인하세요
- 서비스 계정에 시트 편집 권한이 있는지 확인하세요

### "할당량 초과" 오류

- YouTube Data API의 일일 할당량을 초과한 경우입니다
- Google Cloud Console에서 할당량을 확인하고 다음날 다시 시도하세요
